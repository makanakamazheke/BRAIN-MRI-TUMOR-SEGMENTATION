# BRAIN-MRI-TUMOR-SEGMENTATION

README
MRI Reconstruction and Segmentation Pipeline (MATLAB)

I developed a complete MRI processing pipeline in MATLAB, starting from raw k‑space data to the full 2D and 3D spatial brain MRI images. 
This included 2D inverse Fourier reconstruction of a single slice, circular low‑pass and high‑pass k‑space filtering, 3D inverse FFT of a volumetric head dataset, rotation correction of misaligned slices, intensity‑based tumour segmentation on a single slice and the full 3D volume (Dice coefficient against ground truth), and 3D visualisation of the segmented tumour using volume rendering. The code demonstrates core concepts in medical image processing, Fourier transforms, and binary segmentation.
